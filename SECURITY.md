# Security Reporting Policy

We appreciate responsible security research.  
If you identify a potential vulnerability in CTWall, contact us privately so we can investigate and fix it quickly.

## Where to report

Send reports to: **cybergabisoft@gmail.com**

Please include if possible:
- affected component/version,
- clear reproduction steps,
- expected vs actual behavior,
- impact assessment,
- proof-of-concept or logs/screenshots (if available).

## What happens after you report

1. We confirm receipt.
2. We validate and triage the issue.
3. We work with you privately on remediation details.
4. We prepare and publish a fixed release.
5. Public disclosure is coordinated after users have reasonable time to upgrade (typically up to 30 days).

If you submit code or technical support for the fix, we can acknowledge your contribution (or keep you anonymous on request).
Thank you for helping improve CTWall platform security.
